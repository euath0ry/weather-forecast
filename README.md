# Weather API 🌤️

Простой REST API для управления данными о погоде в городах. Позволяет получать, добавлять, удалять и искать информацию о погоде.

## Возможности ✨

- 📋 Получение списка всех записей о погоде
- ➕ Добавление новой записи о погоде
- 🗑️ Удаление записи о погоде
- 🔍 Поиск погоды по названию города

## Технологии 🛠️

- **FastAPI** - современный веб-фреймворк для создания API
- **Uvicorn** - ASGI сервер для запуска приложения
- **Python 3.7+** - язык программирования
